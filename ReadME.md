## RETURN TWO NUMBERS THAT ADD UP TO THE TARGET

This program loops through an array of numbers and select 2 numbers that adds up to the given target.

#### Setup
You must have Node engine (version 16+) installed on your computer and a package installer (yarn or npm).

##### Step 1:
  clone this project

##### Step 2
  yarn (or npm install)

#### Execute program
  on your terminal enter the following commands ```yarn start:dev``` to run the program.

#### Run Test
  On your terminal enter the folllowing commands ```yarn test``` to run the test suite.
